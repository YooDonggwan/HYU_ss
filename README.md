# 스터디 목표 공유 앱 제작 프로젝트
---------------
## 앱 기능 구현  
### 회원가입 기능  
 - 회원가입시 데이터베이스에 같은 아이디가 있는지 확인(**중복검사**)한다.  
 - 비밀번호는 특수문자, 번호, 영문자 또는 한글이 섞여있어야한다.  
 - 본인인증을 위한 메세지 기능 구현  
 - 닉네임 설정 기능    
### 로그인 기능  
 - 로그인 시 **데이터베이스**에 아이디 또는 비밀번호가 정확한지 확인 후, 로그인되게 허용한다.  
##### 사용 프로그램

- 안드로이드 스튜디오
  - Kotlin 사용
- 서버 - Google 클라우드 플랫폼 서버 이용
